Hello

Lately, as Developers, WordPress Power Users or final user we are used to using some Page Builder.

There are tens of Page Builders out there, some bad as Visual Composer some good as Beaver Builder.

Now for a new project that I am starting working on and for as requirement needs a page builder, I did research to know what’s new on this topic.

I found this “relative” new Page Builder called Elementor, and for my surprise, the plugin really nailed it about the UI & UX, is easy to use, it does not fill the content with a shortcodes nightmare, is fast, intuitive and easy to use.

For the next couple of months, I’ll be fully working with Elementor not just using the UI and the default modules, I’ll create custom modules for the project, and I’ll like to document the process.
